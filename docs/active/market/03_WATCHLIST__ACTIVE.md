## Règle
La watchlist n'est pas un portefeuille.

## Idées en suivi

- Actif : SXT
- Type : Crypto
- Horizon : spéculatif / event
- Statut : watchlist seulement
- Catalyseur : à revalider live
- Déclencheur : non validé
- Invalidation : si liquidité / structure se dégradent
- Note : ne pas traiter comme achat implicite

### Idée 2
- Actif : OPN
- Type : Crypto

- Statut : watchlist seulement
- Catalyseur : à revalider live
- Déclencheur : non validé
- Invalidation : si mouvement purement listing / faible liquidité
