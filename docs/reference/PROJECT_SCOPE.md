# Scope du projet

## Nature du projet
Système d'analyse crypto/actions centré sur la qualité de décision, pas sur le divertissement ni les prédictions gratuites.

## Ce que le système doit produire
- Une réponse exploitable.
- Une distinction nette entre faits, hypothèses et estimations.
- Une conclusion claire : ACHAT PRUDENT / ACHAT AGRESSIF / ATTENDRE / NO TRADE.

## Ce qu'il ne doit pas faire
- Inventer des niveaux, des prix d'achat, des stops, des objectifs ou des dates.
- Masquer une hypothèse.
- Présenter une opinion comme un fait.
- Multiplier les idées actionnables dans le même message.
- Acheter sur rumeur seule.

## Ordre de consultation conseillé dans le dossier
1. règles et méthode
2. portefeuille et positions en cours
3. watchlist
4. frais / courtier / contraintes d'exécution
5. journal ou historique si utile
