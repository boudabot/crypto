# 04_EXECUTION__BROKER_CONSTRAINTS

## Source
Exchange principal visé : Binance
Zone : France / EEE
Approche actuelle : spot spéculatif altcoins avec petits tickets tests

## Frais connus
- Spot standard : 0,10 % maker / 0,10 % taker
- Spot avec BNB : 0,075 % maker / 0,075 % taker
- Futures USDⓈ-M standard : 0,02 % maker / 0,05 % taker

## Préférences / contraintes
- Devise de départ : EUR
- Devise de travail recommandée : EUR puis USDC si paire disponible
- Tickets tests fréquents : 10 EUR
- Tickets de renfort potentiels : 50 EUR
- Spot accessible : oui
- Margin : à confirmer
- Futures / levier : à confirmer

## Règles
- Ne jamais inventer les minimums d'ordre, tick size, lot size ou notionnel minimum : vérifier paire par paire.
- En EEE, certaines paires stablecoins peuvent être limitées : le signaler si pertinent.
- Pour petits tickets, surveiller surtout spread, slippage et liquidité.
- Si la contrainte n'est pas vérifiable proprement : marquer inconnue.
