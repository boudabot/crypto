# Design Inspirations

## Purpose
Track useful external UI references that are worth reproducing.

## Template

### Reference N - Name
- Type: URL
- Screen: portfolio / asset page / journal / navigation
- Link: `https://...`
- Copy: exact layout, hierarchy, density, or interaction to reproduce
- Avoid: elements we do not want to copy
- Reason: why this reference matters

## Rule
Keep only annotated links here. If a link is not explained, it should not stay.
