### ACTIVE
Source de verite actuelle, a lire en priorite si le sujet correspond.

Etat courant vivant, handoff court, informations de branche, dernieres decisions.

Historique ou contexte ancien. A conserver mais a ne pas utiliser comme base principale si un fichier ACTIVE existe.
